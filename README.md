# Kifu Notebook

The Kifu Notebook is a web-based app to take a note of Kifu in Shogi.

## Usage

```
$ kifu-notebook
```

## Development of SPA

### Requirements

* Node.js

### Install dependencies

```
$ npm install
```

### Run

```
$ npm start
```

## Development of Command

### Requirements

* Go
* Make

### Build

```
$ make
```
